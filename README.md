# rakawa0825.github.io

This repository is the source for `rakawa0825.github.io`, Naoyuki Hirakawa's public-facing technical portfolio site.

The site focuses on LLM-assisted infrastructure design lifecycle work: structuring fragmented engineering information, preserving uncertainty, keeping human approval visible, and turning professional workflows into reusable artifacts.

## Public-Safe Content Policy

- Do not include private project data.
- Do not include real customer names, real person names, real meeting transcripts, real network diagrams, or real review comments.
- Use only synthetic examples and public-safe descriptions.
- Keep vendor and product references generic unless they are necessary and public-safe.

## Update Workflow

1. Edit Markdown pages and articles locally.
2. Check that all public-facing content is English and public-safe.
3. Run a local safety grep before committing.
4. Commit only after reviewing the rendered Markdown.

## Local Preview

This is a minimal GitHub Pages / Jekyll-compatible site. If Jekyll is available locally, preview with:

```bash
bundle exec jekyll serve
```

No remote, publishing, analytics, comments, or tracking are configured in the initial site.
